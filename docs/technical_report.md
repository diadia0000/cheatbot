# AI 輔助投資詐騙模擬系統
## 技術報告

### 一、背景

近年來，投資詐騙已成為台灣最大宗詐騙類型之一，受害者遍及各年齡層，損失金額動輒數十萬乃至數百萬新台幣。更值得警惕的是，詐騙集團已開始導入 AI 大語言模型技術，以自動化對話取代人工，使詐騙規模得以大幅擴張——一名後台人員即可同時監控數十組詐騙對話，24 小時不間斷運作。
透過開發一套反詐宣導使用的 AI 詐騙模擬系統，來模擬還原真實詐騙話術流程，協助執法人員深入理解詐騙集團的運作模式，進而提升識詐能力與宣導效果。

---

### 二、系統概述

核心功能為模擬投資詐騙的完整話術流程，並提供警方監控視角，可用於：
- **反詐宣導展示**：向民眾直觀呈現 AI 詐騙的實際樣態，提升社會整體防詐意識。
- **話術研究分析**：警方可記錄 AI 在各情境下使用的詐騙話術，作為情資參考。

使用者介面採用民眾最熟悉的 **LINE 通訊平台**，無需額外安裝任何應用程式，完整還原真實詐騙受害情境。

---

### 三、技術架構

#### 3.1 整體架構

系統由三個主要服務元件組成：

| 元件 | 功能說明 |
|------|----------|
| **大語言模型推理引擎** | 本地部署之 LLM（大語言模型），負責依據詐騙劇本與對話歷史生成自然語言回覆 |
| **後端業務邏輯服務** | 核心控制層，負責對話管理、詐騙階段追蹤、記憶檢索、LINE 訊息收發 |

```
LINE 使用者 → LINE Platform → server後端 → LLM 推理引擎
                                                ↕              ↕
                                           對話資料庫      即時股市資料
```

### 3.2 對話歷史記憶機制

為確保詐騙模擬的話術一致性採用以下技術：
- **RAG檢索增強生成**：以語意向量方式儲存對話摘要，供 AI 在需要時檢索過往脈絡，模擬詐騙者「記得你說過的每一句話」的特性。

### 3.3 詐騙階段自動推進

系統將完整的殺豬盤詐騙流程切分為**五個階段**，AI 會依據對話輪次與語意判斷，自動推進至下一階段，且階段只進不退，符合真實詐騙的不可逆特性：

| 階段 | 名稱 | 核心目的 |
|------|------|----------|
| 第一階段 | 破冰建立關係 | 噓寒問暖，讓受害者感覺「對方是真誠的朋友」 |
| 第二階段 | 展示投資實力 | 引用真實股市行情曬獲利，建立專業可信形象 |
| 第三階段 | 試探資金意願 | 探問閒置資金，降低入門門檻 |
| 第四階段 | 製造急迫感 | 以「限額名額」「截止時間」施壓，消除猶豫 |
| 第五階段 | 引導匯款 | 提供假帳號，完成詐騙收網 |

透過解析 LLM 的內部推理標籤（`<thought>`）來判斷當前應進入哪個階段，而非單純以對話輪次決定，使話術推進更加自然。

### 3.4 即時股市資料整合

為還原詐騙集團「用真實數據騙人」的手法，系統串接 **FinMind 台股即時行情 API**，在詐騙進入第二階段後，自動將當日實際股價資訊注入LLM。

AI 能據此說出如：「今天台積電又漲了，我早就說這支有機會」等話術，且數據來源為真實市場，提升模擬的真實感。

### 3.5 假照片生成
透過外部圖像生成 API（如 DALL·E）製作假照片，模擬詐騙者在社交平台上曬出投資獲利證明或是學員好評等資訊，增加詐騙話術的說服力。

---

### 4. 系統功能示意

以下為典型演練流程：

```
受害者透過 LINE 傳送：
  「你好，我是在 FB 上看到老師你有在帶投資，有點興趣」

↓ 系統收到訊息後：
  1. 儲存對話至資料庫
  2. 查詢當日台股行情（第二階段後啟用）
  3. 呼叫 LLM 生成符合當前詐騙階段的回覆
  4. 解析 AI 內部推理，判斷是否需推進至下一階段
  5. 透過 LINE Reply API 回傳多段訊息

↓ AI 回覆示例（第一階段）：
  「哈哈你好！你是在哪邊看到關於我的資訊，你平常有在關注股票嗎？」
```

## 結語

隨著 AI 技術普及，詐騙集團已不再需要大量人力即可執行大規模詐騙攻勢。透過以技術對抗技術的思路，開發出可實際應用於宣導的工具，期望藉此縮短大眾對新型詐騙手法的理解落差。
